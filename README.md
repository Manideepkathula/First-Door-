# FirstDoor 🚪

The survival guide for international students arriving in Boston.

Built by Manideep Kathula — an international student from Warangal, India who went through every challenge this app covers.

## What it covers
- 🏠 Housing — neighborhoods, rent ranges, tenant rights
- 🛒 Food — South Asian groceries, Indian restaurants, halal/veg options
- 💳 US Systems — credit score, bank accounts, health insurance, SSN
- 🚇 Transit — MBTA guide, Charlie Card, Logan Airport arrival
- 📋 Visa — CPT vs OPT, F-1 rules, key deadlines
- 🆘 Emergency & Legal — tenant rights, healthcare costs, scam protection

## Tech Stack
- React + Vite
- React Router DOM
- Lucide React (icons)
- Claude API (AI assistant)
- Deployed on Vercel

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push to GitHub
2. Import repo on vercel.com
3. Deploy — zero config needed (vercel.json handles routing)
